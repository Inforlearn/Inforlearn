**actor_add_contact** (owner, target)


Xin kết bạn/đăng ký nhận tin từ owner tới target

  Tham số:
    * owner - nick của bạn
    * target - nick người muốn kết bạn

  Trả lời:

  ::

     {
       'status': 'success',
       'rv': {
                'relation': {
                              'owner': 'aloneroad@inforlearn.com',
                              'relation': 'contact',
                              'target': 'admin@inforlearn.com'
                            }
              }
     }
