Welcome to NoPix

Your own pix sharing service

NoPix is a free open source application to share you pictures. It works like ImageShack. So you can use this app for your blog, forum,... You have to sign if you want to test the history feature.

Do not use directly this site but install you own version. You can download the sources here (2010-01-11). Set the config.py to make this site totally private or chose the picture size and more for the thumbnails.
